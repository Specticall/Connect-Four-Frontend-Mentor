RULES

OBJECTIVE

Be the first player to connect 4 of the same colored discs in a row (either vertically, horizontally or diagonally).

HOW TO PLAY

  1. 1. Red goes first in the game
  2. 2. Player's must alternate turns, and only one disc can be dropped in each turn
  3. 3. The game ends when there is a 4-in-a-row or a stalemate
  4. 4. The starter on the previous game goes second on the next game
